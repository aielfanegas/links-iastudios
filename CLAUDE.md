# LINKS.IASTUDIOS

Link-hub estilo home-screen iOS que distribuye los links de Samuel (proyectos, redes, contacto). *Deliverable* propio, no un "mundo" operativo.

## Stack
- HTML5 + CSS3 (glass-morphism, flexbox) + **JavaScript vanilla** (sin framework, sin build)
- Features: drag-to-reorder (pointer-capture), modal carpeta, QR via API externa, vCard descargable (`contacto.vcf`)
- `manifest.webmanifest` → instalable (PWA mobile-first)

## Editar
Abrir `index.html` directo. No hay paso de build. Logos/fondos = assets en la raíz.

## Convenciones
- Sin emojis en UI. Iconos = imágenes/SVG.
- Mantener sincronizados los links con los proyectos reales (GOBBS, NINOOX, MiRiñón, redes).
